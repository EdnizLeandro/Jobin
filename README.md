# Jobin
Jobin: Solução Criativa para Jovens em Transformação. Este projeto apoia jovens na superação da inatividade, por meio de estratégias baseadas no Human-Centered Design da ONU. Integrando educação, empreendedorismo e inclusão social, fortalecendo suas habilidades, auxiliando-os a retomar a seus estudos e iniciar uma trajetória de trabalho autônomo.
